# Brightline Motion - WALKED (2025)

Robotics integrator, $61M rev. LOI signed at 10.1x, walked at confirmatory.

What killed it: ERP + MES integration scoped at 6mo, vendor quotes came back
14-18mo. Synergy case evaporated. $1.8M dead-deal cost.

Lesson: integration risk is the silent killer. Any target with bespoke internal
systems gets an integration-complexity score BEFORE LOI. Weight this heavily.
